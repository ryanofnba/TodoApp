Todo App

This app will include the following features:

  • Adding a todo item
  • Editing a todo item
  • Deleting a todo item

Tools required to build this app:

  • HTML 5 for templating
  • Javascript, with the use of JQuery for DOM manipulation
  • CSS 3, with the uses of Bootstrap for styling
  • Version control with Git using Github

  • HTML 5 Documentation
